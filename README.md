# 🤬 转人工！

循环播放“转人工”助你跳过 AI 客服

一个使用 Vite 8 + React + TypeScript + react95 构建的单页应用。页面中央有一个按钮，点击后无限循环播放"转人工"的中文语音。

## 技术栈

- [Vite 8](https://vite.dev/) — 构建工具
- [React 19](https://react.dev/) + TypeScript
- [react95](https://github.com/react95-io/React95) — Windows 95 风格 UI 组件库
- [Web Speech API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API) — 中文语音合成

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
